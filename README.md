# Zootopia API Explorer 🦊

*English version below / Englische Version unten*

Dieses Projekt generiert eine benutzerdefinierte Website, die Tierinformationen anzeigt, welche basierend auf Benutzereingaben dynamisch von der API Ninjas Animals API abgerufen werden.

## Installation

1. Klone dieses Repository auf deinen lokalen Rechner.
2. Installiere die benötigten Abhängigkeiten mit pip:
   `pip install -r requirements.txt`
3. Erstelle eine `.env`-Datei im Hauptverzeichnis und füge deinen API Ninjas Schlüssel hinzu:
   `API_KEY=dein_api_schlüssel_hier`

## Nutzung

Um dieses Projekt zu nutzen, führe das Hauptskript in deinem Terminal aus:
`python animals_web_generator.py`

Du wirst aufgefordert, den Namen eines Tieres einzugeben. Das Skript ruft die Daten ab und generiert eine `animals.html`-Datei mit den Ergebnissen. Öffne diese Datei in deinem Webbrowser, um dein individuelles Tier-Repository anzusehen!

# Zootopia API Explorer 🦊 (English)

This project generates a custom website displaying animal information fetched dynamically from the API Ninjas Animals API based on user input.

## Installation

1. Clone this repository to your local machine.
2. Install the required dependencies using pip:
   `pip install -r requirements.txt`
3. Create a `.env` file in the root directory and add your API Ninjas key:
   `API_KEY=your_api_key_here`

## Usage

To use this project, run the main script in your terminal:
`python animals_web_generator.py`

You will be prompted to enter the name of an animal. The script will fetch the data and generate an `animals.html` file containing the results. Open this file in your web browser to view your custom animal repository!
